1st Sault Ste. Marie Scouts — complete replacement website

This package is a ground-up static website with no Scouts Canada logo or other
image asset required.

Files:
- index.html
- about.html
- sections.html
- parents.html
- volunteer.html
- join.html
- contact.html
- beavers.html
- cubs.html
- scouts.html
- venturers.html
- rovers.html
- css/styles.css
- js/main.js

ScoutsTracker:
All ScoutsTracker buttons currently point to:
https://scoutstracker.ca/

Change this URL in the HTML files if your group has a direct ScoutsTracker
section or group address.

Contact form:
The contact form is a front-end demonstration only. Connect it to your chosen
form provider, email service or backend before publishing it as a working form.

Deployment:
Upload the complete contents of this folder to the root of your GitHub Pages
repository. This package does not require Firebase or a build step.

Branding:
The site uses text-only group branding. No Scouts Canada logo is included.
